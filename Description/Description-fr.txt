Pour mon nouveau projet, je teste aujourd'hui brièvement le vidéoprojecteur LED M2W de ViewSonic.

Avec 1700 lumens LED, le dernier modèle est le plus lumineux de ses trois collègues M2 :

- M2 : 1200 lumens LED
- M2e : 1000 lumens LED 
- M2W : 1700 lumens LED
