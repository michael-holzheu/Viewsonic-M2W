Для своего нового проекта я сегодня кратко тестирую светодиодный лучемет ViewSonic M2W.

С 1 700 светодиодными люменами, последняя модель является самой яркой из трех коллег M2:

- M2: 1200 светодиодных люмен
- M2e: 1000 светодиодных люмен 
- M2W: 1700 светодиодных люмен

Маркеры прыжков:

00:01 Распаковка Beamer
06:01 Включить
07:20 Настройка ориентации
09:28 Настройка беспроводной локальной сети
13:58 Зеркальное отображение экрана с помощью Air Play (пробная версия)
17:07 Подключение USB-C
19:25 Просмотр фильма (тест)
20:50 Настройка формата изображения
21:20 Изображение дергается, кадры потеряны

Настройка записи:

Для съемки видео будет использоваться Samsung Galaxy S22 Ultra, а аудио *не* будет редактироваться впоследствии.

Предложения по улучшению:

- 08:44 Прогресс бар при настройке горизонтального/вертикального положения.

Использована музыка:

- Успех всей команды (WinnieTheMoog).
